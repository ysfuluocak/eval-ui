# Eval-Ui

Değerlendirme formu uygulaması (React + Ant Design)

## Özellikler
- Çok sayfalı form yapısı
- LocalStorage ile taslak kaydetme
- Ant Design komponentleri
- Responsive tasarım

## Kurulum
```bash
npm install
npm start
```
